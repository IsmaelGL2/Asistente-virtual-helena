# 🗣️ Asistente Virtual Helena

**Helena** es un asistente virtual desarrollado en Python que reconoce comandos de voz y responde con audio, permitiendo realizar tareas cotidianas como abrir aplicaciones web, buscar información, consultar la hora, reproducir música y más.

## 🚀 Funcionalidades

- 🎤 Reconocimiento de voz en español
- 🔊 Respuesta por voz (text-to-speech)
- 📅 Consulta del día y la hora actual
- 🌐 Apertura de YouTube y Google
- 📖 Búsquedas en Wikipedia
- 🔍 Búsquedas rápidas en internet
- 🎵 Reproducción de videos en YouTube
- 📈 Consulta del precio de acciones (Apple, Amazon, Google)

## 🛠️ Tecnologías utilizadas

- `pyttsx3` – síntesis de voz
- `speech_recognition` – reconocimiento de voz
- `pywhatkit` – búsquedas y reproducción en YouTube
- `wikipedia` – acceso a información de Wikipedia
- `yfinance` – consulta de acciones en bolsa
- `webbrowser` – apertura de enlaces
- `datetime` – manejo de fecha y hora
